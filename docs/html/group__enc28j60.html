Detailed Description

 #include "net/enc28j60.h"
Overview
This driver provides initialization and transmit/receive functions for the Microchip ENC28J60 10Mb Ethernet Controller and PHY. This chip is novel in that it is a full MAC+PHY interface all in a 28-pin chip, using an SPI interface to the host processor.
