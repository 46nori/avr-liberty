00067 //----- Single Channel Functions ----------------------------------------------
00068 // The following 4 functions are used for controlling a single channel on the
00069 // a2d converter.  If you are only using 1 channel, then these functions are
00070 // faster.  If you are using 2 or more channel, it is faster to use the All
00071 // Channel Functions.  This is because to convert all the channel only requires
00072 // one command to the device, and then a read of the all the channel.  To read
00073 // two channel individually requires two commands to the device, and two reads.
00074 // Therefore using the All Channel Functions for even just 2 channels is faster
00075 // and more effificient.
